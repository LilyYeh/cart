# 用 laravel7.0 + vuejs2 做一個小購物車

#### 環境建置
##### 步驟1. 安裝 node 和 composer
```
> cd cart;
> npm install;
> composer install;
```

##### 步驟2. 新增 .env 檔
```
> cp .env.example .env
```

##### 步驟3. start docker
```
> cd docker;
> docker-compose up -d;
```

##### 步驟3. 瀏覽
http://localhost/cart

#### 開發工具
1. laravel 7.0
2. vue.js 2
